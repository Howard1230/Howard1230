	統計資訊學系的訓練，讓我熟悉各類分析工具與程式語言，如 R、Python、Java、Excel 及 SPSS，並能夠獨立完成資料分析、視覺化、模型建構與報告撰寫等能力。這些技能讓我能有效處理實務問題並提出有根據的決策建議。
